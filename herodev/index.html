Install

Note that only Linux and MaxOSX is supported right now, and not extensively tested at all (more an alpha release really)


$ curl https://s3.amazonaws.com/leonkenneth/herodev/setup | bash

And follow instructions.

What is it?

It allows for a F5-driven development workflow on Heroku.

How to use it?

It's still an early release, but for now:


$ cd myherokuapp/
$ heroku dev:init # Just once
$ # It will create a commit you'll have to review then push
$ git push heroku master
$ # At the beginning of your coding session just
$ heroku dev:watch
